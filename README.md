# React Structure Sample

This workspace contains a minimal React project structure scaffolded for development with Vite.

Commands:

```bash
npm install
npm run dev
```

Structure:

- `public/` — static index.html
- `src/` — source files
  - `components/` — reusable components
  - `pages/` — page-level components
  - `App.jsx`, `main.jsx`, `index.css`
